# 坦克大战 (Tank Battle)

一个纯 HTML/CSS/JavaScript 实现的经典坦克大战游戏。

## 操作方式

| 按键 | 功能 |
|------|------|
| ↑↓←→ | 移动坦克 |
| 空格 | 发射子弹 |
| Enter | 开始/暂停 |
| R | 重新开始 |

## 游戏规则

- 消灭所有敌方坦克即可过关，共3关
- 保护下方基地（旗帜标记），基地被摧毁则游戏结束
- 玩家有3条生命
- 每关敌人数量递增

## 技术栈

- HTML5 Canvas
- 纯 JavaScript（无框架依赖）
- CSS3

## 运行

直接在浏览器中打开 `index.html` 即可。
